# QGIS Files

This folder contains QGIS project files used to visualize traffic volumes and congestion levels on a map.

## Files
- `Volume_Map_2023_2024.qgz`: A QGIS project showing aggregated volume changes across links for July to September 2023 vs 2024.
- `CommonLinks_2023_2024.qgz`: QGIS layer containing common traffic links used across multiple analyses.

These files were created using link-level geospatial data from the City of Gold Coast.
